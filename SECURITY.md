# Security Policy

## Supported Versions

As bookistudios-ai doesn't provide an official release yet, please use the latest version for the security updates.
Currently, we have two branches to maintain:
* main branch for bookistudios-ai 2.x
* main-1.x branch for bookistudios-ai 1.x 

## Reporting a Vulnerability

Please go to https://github.com/bytedance/bookistudios-ai/security to report the vulnerability you find.
